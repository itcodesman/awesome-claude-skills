# 专业协作、合规流程与费用结构

## 一、专业团队分工

| 角色 | 职责 | 注意事项 |
|------|------|---------|
| 美国持牌律师（Attorney） | 起草信托协议，必须在执业辖区内服务 | 律师仅能在其执业州提供法律服务 |
| 注册会计师（CPA） | 年度税务申报（Form 1041等）、税务规划 | 需熟悉跨境信托税务 |
| 家族办公室（FO） | 架构导航、多辖区整合、投顾执行 | 角色为顾问而非决策方 |
| 信托公司（Trust Company） | 担任受托人，执行信托协议 | 银行系受限多，独立信托公司灵活性更高 |
| 私人信托公司（PTC） | 委托人家族自设受托人 | 需自建合规团队 |
| 移民顾问 | 护照获取、身份规划 | 需具备目标国合法资质 |

## 二、KYC / AML 合规材料清单

信托设立前须准备以下文件：

- [ ] 护照及身份证明文件（含新护照，如适用）
- [ ] 境外地址证明（水电账单、银行账单，地址需一致）
- [ ] 外国税号（Tax Identification Number）
- [ ] 资金来源证明文件
  - 股权转让协议
  - 企业财务报表
  - 房产买卖合同
  - 薪酬记录 / 年度税务申报表
- [ ] 受益人信息（真实身份关系，须与信托协议一致）
- [ ] 已婚人士：配偶签署的财产放弃声明（Spousal Consent）

## 三、信托设立全流程

### 阶段一：需求分析与架构设计

- 明确核心目标（资产保护 / 传承 / 节税 / CRS规避）
- 选择司法辖区（美国 / 库克群岛 / 欧洲基金会）
- 确定治理结构（各角色人选）
- 初步方案评审

### 阶段二：法律文件起草

- 信托协议（Trust Deed）起草
- 意愿书（Letter of Wishes）起草
- 子信托设计与文件化
- 配偶声明（如适用）

### 阶段三：KYC / AML 合规审查

- 提交委托人身份信息
- 提交资金来源证明
- 信托公司内部合规审核
- 必要时律所出具意见书

### 阶段四：账户开立与资产置入

- 申请信托EIN（税号）
- 以信托名义在券商/银行开户
- 资金汇入信托账户
- 保单/不动产等资产产权转移登记

### 阶段五：年度维护与申报

- 年度税务申报（Form 1041等）
- 资产状况更新
- 意愿书定期审阅与更新
- 架构定期评估（建议每3–5年）

## 四、费用结构体系

| 费用项目 | 金额区间 | 说明 |
|---------|---------|------|
| 简单架构设立费 | ~3万美元 | 无监察人、无分配顾问的基础架构 |
| 复杂架构设立费 | 5–6万美元 | 多重角色、多子信托、多资产隔离 |
| 超大型架构 | 不超过20万美元 | 极复杂家族架构 |
| 信托公司年度维护费 | 数千至1–2万美元 | 取决于服务规模；纯金融资产时极低 |
| 欧洲基金会年费 | 资产规模1%以上 | 防穿透能力强，但维护成本高 |
| 护照获取费用 | 30–50万欧元 | 斯洛文尼亚/爱尔兰，含政府费、中介费、律师费 |
| CPA年度税务申报 | 视复杂度而定 | 多表格申报时费用上升 |

## 五、常见问题与风险提示

### Q：信托成立后委托人还能控制资产吗？
**A**：不可撤销信托中，委托人不能直接控制。但可以通过：
1. 担任分配顾问（对分配行使否决权）
2. 通过投顾角色指令受托人执行特定投资
3. 通过意愿书动态指引受托人行为
直接控制会带来信托被"击穿"风险，专业顾问会设计合规的间接控制路径。

### Q：中国人能设立美国信托吗？
**A**：可以。中国籍人士可在美国南达科他州等辖区设立不可撤销信托，以信托EIN开户持有美股。但需注意：
- 须完成完整的KYC/AML审查
- 建议同步规划护照/身份策略以切断CRS信息链
- 资金来源须合规证明

### Q：信托和遗嘱有什么区别？
**A**：信托在委托人在世时即生效，资产直接转移至信托名下，身故时无需走遗嘱认证程序（Probate）。遗嘱信托须等委托人身故后才生效，且需经过法院认证，效率较低、信息可能公开。
