# 国际税务合规体系

## 一、CRS（共同申报准则）

### 基本原理

- **全称**：Common Reporting Standard，OECD发布，2014年推行
- **机制**：参与国金融机构识别非本国税务居民账户，向账户持有人所在税务居民国**自动**报送账户信息
- **信托CRS穿透**：参与CRS国家的信托须识别并报告委托人/受益人/监察人等实际控制人

### 信托在CRS框架下的报告义务

| 角色 | CRS下的报告要求 |
|------|---------------|
| 委托人 | 若保留控制权，须作为"实控人"被报告 |
| 受益人 | 若为确定性受益人，须被报告 |
| 监察人 | 拥有重大影响权力，须被报告 |
| 受托人 | 机构受托人本身为申报义务方 |

### 合法规避CRS穿透的路径

1. **使用非CRS参与国辖区（首选：美国）**：无CRS申报义务
2. **以非中国籍身份设立**：在CRS参与国辖区以第三国国籍设立，切断与中国信息链
3. **欧洲基金会结构**：基金会作为独立法人，创始人权益中断，CRS报告链断开

## 二、FATCA（美国海外账户税收合规法案）

- **全称**：Foreign Account Tax Compliance Act（2010年立法）
- **机制**：要求全球金融机构识别并报告**美国人（US Persons）** 账户信息至IRS
- **适用人群**：美国公民、美国税务居民、美国绿卡持有人
- **对信托的影响**：若委托人或受益人为美国人，信托可能触发FATCA报告义务
- **与CRS关键区别**：CRS是双边/多边交换；FATCA是单向报美国

---

## 三、美国遗产税（Estate Tax）

> ⚠️ **高净值人士的重大风险盲区**：非美国税务居民的美国遗产税免税额极低（仅6万美元），在美国持有股票、房产须高度警惕。

| 项目 | 说明 |
|------|------|
| 适用对象 | 美国公民及美国税务居民：对全球资产征税；非居民：仅对美国境内资产征税 |
| 税率 | 超出免税额部分最高 **40%** |
| 2024年免税额（美国公民/居民） | 约 1,361万美元/人；已婚夫妇约 2,722万美元 |
| **非居民免税额** | **仅6万美元**（与公民/居民相差200倍以上） |
| 信托规避路径 | 非美税务居民设立不可撤销信托，信托资产脱离个人遗产 |

### 非居民美国遗产税风险举例

- 在IBKR账户直接持有美股：遗产税敞口 = 账户价值（扣除6万美元）× 最高40%
- 解决方案：通过不可撤销信托（EIN开户）持有美股，资产脱离个人遗产

## 四、赠与税（Gift Tax）

- 美国联邦赠与税与遗产税**共享终身免税额**
- 年度赠与免税额：约 **1.8万美元/人**（2024年）
- 非美国人将资产置入信托不视为赠与应税事件（须结构合规）

---

## 五、信托收益的税务处理

| 场景 | 税务处理 |
|------|---------|
| 信托收益未分配至委托人 | 不触发委托人个人纳税义务 |
| 信托为美国境内设立 | 信托本身为独立纳税主体（EIN） |
| **委托人信托（Grantor Trust）** | 税务上视委托人为所有人，收益合并至委托人个人申报 |
| **非委托人信托（Non-Grantor Trust）** | 信托独立报税，适用信托税率表 |

## 六、合规申报义务汇总

| 申报表格 | 适用场景 | 申报机构 |
|---------|---------|---------|
| **Form 3520** | 美国人接受外国信托赠与 | IRS |
| **Form 3520-A** | 外国信托年度申报 | IRS |
| **FinCEN 114 (FBAR)** | 美国人持有境外账户 >1万美元 | FinCEN |
| **Form 8938** | 美国人持有境外金融资产（FATCA申报） | IRS |
| **Form 1041** | 信托年度所得税申报 | IRS |

## 七、税务居民身份管理

| 概念 | 说明 |
|------|------|
| 税务居民认定 | 通常以居住天数（183天规则）或永久居住地为标准 |
| 全球征税原则 | 美国、澳大利亚等国对本国税务居民实行全球征税 |
| 属地征税原则 | 香港、新加坡仅对境内所得征税，对海外所得免税 |
| 退出税（Exit Tax） | 美国公民/永久居民放弃身份时，须申报未实现资产增值并缴税 |

### 新加坡税务居民（NRA视角）

- 新加坡采用属地征税，海外收入一般免税
- 新加坡参与CRS，IBKR账户信息会向账户持有人税务居民国申报
- W-8BEN申报新加坡税居：IBKR仅向IRAS（新加坡税务局）报告，不向中国报告
- 股息预扣税：美国来源股息对非居民预扣30%（或根据税收协定降低）
- **资本利得**：美国非居民出售美股一般免征联邦资本利得税（IRC §871(a)(2)）
