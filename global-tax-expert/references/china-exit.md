# 中国税务居民身份退出规划
> 核心矛盾：客户**不愿放弃国内社保和房产**，但希望**切断中国税务居民身份**
> 好消息：这两件事**可以并存**——持有房产和缴纳社保，不等于中国税务居民

---

## 一、中国税务居民认定的两种路径（关键！）

### 路径一：有"住所"（domicile）的人

> 《个人所得税法》第1条：在中国境内有住所……为居民个人

**"住所"的法律定义**（实施条例第2条）：
> 因户籍、家庭、经济利益关系而在中国境内**习惯性居住**

关键词是"**习惯性居住**"，不是"拥有住房"。

| 情况 | 是否认定有"住所" |
|------|---------------|
| 户籍在中国，长期在境外生活 | **视具体情况而定**，SAT实务中会考察实际居住行为 |
| 拥有国内房产但长期定居新加坡 | **通常不认定为有住所**（习惯性居所在境外） |
| 家庭主要成员在中国，本人常住境外 | **存在争议**，是SAT审查重点 |
| 经济利益中心明显转移至境外 | **倾向于非居民**认定 |

> ⚡ **关键结论**：在中国拥有房产本身**不等于**拥有"住所"。"住所"取决于习惯性居住地，不是房产所有权。

---

### 路径二：无住所但每年居住≥183天

> 《个人所得税法》第1条：……或者无住所而一个纳税年度内在中国境内居住累计满183天的个人

**183天规则的精确计算**（2019年第35号公告）：
- 以自然年（1月1日-12月31日）为纳税年度
- 出入境当天**均计入**在华天数（按半天各计算）
- 在中国境外停留时间不纳入计算

**如何安全控制在183天以下**：
- 建议全年在中国实际天数控制在**150天以内**（留有缓冲）
- 完整记录出入境记录（护照盖章 + 订机票/酒店记录）
- 如果某年意外超过183天，当年仍是中国税务居民，但不影响下一年重新规划

---

## 二、社保缴纳不触发税务居民身份（重要！）

### 法律层面的分离
- 社会保险（养老/医疗/工伤/失业/生育保险）由**人力资源和社会保障部**管辖
- 个人所得税由**国家税务总局**管辖
- 两套体系**相互独立**，缴纳社保**不是**税务居民身份的认定依据

### 社保缴纳的实际影响
| 影响 | 是否存在 |
|------|---------|
| 缴纳社保 → 被认定为中国税务居民 | **否**，无直接法律关联 |
| 缴纳社保 → 产生中国来源收入义务 | **否**（社保是缴费，非收入） |
| 社保账户信息被纳入CRS申报 | **否**，社保账户不在CRS申报范围 |
| 社保账户用于追踪金融资产 | 理论上可能被调查时参考，但不是常规申报渠道 |

> ✅ **结论**：继续缴纳国内社保，**不会直接触发中国税务居民身份**，更不会导致境外金融资产被追税。

---

## 三、持有国内房产的税务影响

### 房产持有本身
- 如前文所述，持有房产 ≠ 有"住所" ≠ 税务居民
- 但房产的**存在**可能在SAT争议认定时被作为"习惯性居住"的佐证

**建议**：建立记录证明实际居住在新加坡（如SG的水电账单、银行账单、学校记录等），以反驳SAT可能的认定。

### 房产产生的税务义务（非居民仍有）
即使成功切断中国税务居民身份，持有国内房产仍有以下义务：

| 税务事项 | 税率/规则 |
|---------|---------|
| **租金收入** | 20%个人所得税（财产租赁所得），通常由承租人代扣 |
| **出售房产差价** | 20%个人所得税（财产转让所得），按"售价-原值-合理费用"计算 |
| **房产税** | 按租金收入12%或房产原值1.2%计征（各地执行不同） |
| **增值税** | 出售时按规定缴纳（个人住宅通常有豁免） |

> 非居民持有中国不动产，**仅对来源于中国的房产相关收入负税**，境外金融资产不受影响。

## 四、5年规则（针对外籍人士的历史规定）

> ⚠️ 注意：5年规则**对中国国籍人士并不适用**，以下内容仅供了解对比

《个人所得税法》修订前（2018年前）有"5年规则"：无住所外籍人士在中国连续居住满5年后，才须就全球收入缴税。

2018年修订后，183天规则对所有人（包括外籍人士）统一适用，5年规则已废除。

中国国籍人士的认定更为严格：由于中国对本国公民采用属人管辖原则的倾向，中国籍人士在境外的税务居民主张可能受到更多审查。

## 五、CRS：与税务居民身份脱钩的理解

### CRS上报路径（精确）
```
新加坡/香港金融机构
    ↓ 调查账户持有人税务居民状态
    ↓ 若账户持有人自报或被认定为"中国税务居民"
    ↓
上报至IRAS（新加坡）/ IRD（香港）
    ↓
自动交换至中国SAT
```

### 关键：CRS看的是"税务居民身份"，不是"国籍"

| 账户持有人情况 | CRS是否上报至中国 |
|-------------|----------------|
| 中国国籍 + 中国税务居民 | ✅ 上报 |
| 中国国籍 + **非**中国税务居民（SG税务居民） | ❌ **不上报至中国**（上报至SG） |
| 已入籍新加坡 + 非中国税务居民 | ❌ 不上报至中国 |

> ⚡ **核心结论**：切断中国税务居民身份 = 合法阻断SG/HK账户信息流向中国SAT。这是最根本的解决路径。

### CRS不覆盖的账户类型（合法豁免）
根据OECD CRS标准第VIII节C款：
- **退休/养老账户**（符合条件的政府养老金计划）：中国境内社保养老账户**不在CRS申报范围内**
- **低余额账户豁免**：个人账户余额 < USD 50,000（部分金融机构适用）
- **特定保险合同**：无现金价值的纯消费型保险
- **不动产本身**：房产**不是**金融账户，不通过CRS申报

> ✅ **结论**：国内社保账户和房产本身，都不在CRS申报范围内，不会通过CRS渠道被上报。

## 六、切断中国税务居民身份的操作路径

### Step 1：确保每年在中国天数 < 183天
- 目标：150天以内（留出缓冲）
- 工具：精确日历记录，出入境记录留存

### Step 2：建立"习惯性居所在境外"的证据链
| 证据类型 | 具体做法 |
|---------|---------|
| 居住证明 | SG长期居住地址的水电费账单、银行对账单 |
| 工作/生活证明 | EP雇主合同、SG注册公司地址、SG子女入学 |
| 经济利益中心 | 主要银行账户在SG、投资账户在SG/海外 |
| 家庭成员状态 | 配偶/子女是否同在SG居住 |

### Step 3：在银行开户时正确申报税务居民状态
- 在SG/HK开户时，如实申报为"新加坡税务居民"（而非中国税务居民）
- CRS申报将按申报的税务居民国处理
- 金融机构可能要求提供税务居民证明（Tax Residency Certificate，TRC）

### Step 4：获取新加坡税务居民证明（TRC）
- 向IRAS申请Certificate of Residence
- 证明在SG满足税务居民条件
- 可在银行开户、CRS申报时出具

### Step 5（可选）：处理国内金融账户
- 国内银行账户、证券账户：关注其是否因"非居民"身份而须调整申报
- 建议将国内账户余额控制在必要水平，减少境内金融账户敞口

## 七、SAT追税风险评估与防御

### 中国SAT可能的追税路径
1. **CRS接收信息**：通过CRS收到境外账户信息后，与税务申报比对
2. **SAFE外汇管制联动**：大额境外汇款触发SAFE报告，可能被SAT关注
3. **举报/稽查**：第三方举报或常规稽查
4. **离境申报**（目前尚未全面实施）：部分草案讨论过对高净值人士离境税，目前未正式立法

### 防御策略
| 风险点 | 防御措施 |
|--------|---------|
| CRS信息比对 | 切断中国税务居民身份（根本解法） |
| 大额境外汇款 | 控制单次金额，了解SAFE$50,000年度购汇限额规则 |
| 国内收入申报 | 按时如实申报国内来源收入（租金等），保持合规底色 |
| 税务记录 | 保留出入境记录5年以上，证明非居民身份 |

## 八、关键法规依据

| 法规 | 内容 |
|------|------|
| 《个人所得税法》第1条 | 居民个人定义（住所 + 183天双路径） |
| 《个人所得税法实施条例》第2-4条 | "住所"的精确定义；183天计算规则 |
| 财税〔2019〕35号 | 非居民个人所得税扣缴规定 |
| 国家税务总局公告2019年第35号 | 居民个人全球收入申报规定 |
| 《金融账户涉税信息自动交换管理办法》（2017年） | CRS在中国的实施细则；金融账户范围定义 |
| 中国-新加坡税收协定第4条 | 双重居民身份Tie-breaker规则 |
| OECD CRS标准第VIII节C款 | 豁免账户类型（社保账户、退休账户等） |
| 国家外汇管理局：个人外汇管理办法 | $50,000年度购汇限额；境外汇款申报 |
