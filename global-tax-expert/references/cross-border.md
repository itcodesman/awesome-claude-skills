# CRS/FATCA 敞口分析与资产结构化
> 核心问题：中国SAT能通过什么渠道获取你的境外资产信息？
> 核心答案：**主要是CRS**，而切断中国税务居民身份是最根本的应对策略

## 二、CRS精确机制解析

### 什么账户被申报

**申报范围（金融账户）**：
- 存款账户（银行账户、活期/定期存款）
- 托管账户（证券账户、基金账户）
- 特定人寿保险合同（有现金价值）
- 年金合同

**不在CRS申报范围内**：
- 中国境内社保/公积金账户（政府养老金计划豁免）
- 中国境内不动产（房产不是金融账户）
- 无现金价值的纯消费型保险
- 账户余额 < USD 50,000的个人存款账户（低余额豁免，部分机构适用）

### 触发申报的条件
金融机构须收集并上报的账户持有人信息：
1. 账户持有人的**税务居民国**（通过自我申报+尽职调查确认）
2. 账户余额/价值
3. 账户产生的收入（利息、股息、出售收益等）

> ⚡ **关键**：CRS看的是**税务居民身份**，不是国籍。切断中国税务居民身份后，SG/HK账户不再上报至中国。

### 中国实施CRS的法规依据
- **《金融账户涉税信息自动交换管理办法》**（国家税务总局、财政部等五部委，2017年）
- 中国2018年开始首次接收来自其他国家的CRS数据
- 目前已与100+国家/地区建立自动交换关系

## 三、美国不参与CRS的战略意义

### 核心事实
美国是**全球主要经济体中唯一不参与CRS的国家**。

| 信息流向 | 机制 | 对中国SAT的可见性 |
|---------|------|----------------|
| SG账户 → 中国SAT | CRS自动交换 | ✅ 可见（若你是中国税务居民） |
| HK账户 → 中国SAT | CRS自动交换 | ✅ 可见（若你是中国税务居民） |
| **美国账户 → 中国SAT** | **无CRS渠道** | **❌ 不可见（通过常规CRS渠道）** |

### 实际应用
- 在**IBKR美国主体**（Interactive Brokers LLC）持有账户：不会通过CRS上报至中国
- 在**美国银行**（如Charles Schwab国际账户）持有资产：同上
- 美国对境外账户持有人（非居民外国人）的信息申报：主要通过FATCA单向上报（美→各国），但**中国向美国的信息交换极为有限**

### 重要注意事项
- 若未来成为美国税务居民（绿卡/长期居住），美国账户会触发FATCA全球申报
- 目前对于非美税务居民的中国公民，在美国持有账户是合法的信息隔离工具
- 不代表不受任何监管——反洗钱（AML）、KYC要求仍然适用

## 四、持股结构选择：个人直持 vs. 法人实体

### 结构对比矩阵

| 持股结构 | CRS申报 | 穿透风险 | 税务效率 | 设立成本 | 管理复杂度 |
|---------|--------|---------|---------|---------|----------|
| **个人直持（SG账户）** | 申报至SG（若切断中国居民身份则不至中国） | 低 | 高（SG免税） | 零 | 低 |
| **个人直持（美国账户）** | 不通过CRS申报至中国 | 最低 | 高（NRA状态无资本利得税） | 零 | 低 |
| **新加坡家族办公室（SFO）** | 以法人名义申报，中国个人信息相对隔离 | 中（需满足实质要求） | 最高（13O/13U免税） | 高（设立+合规） | 高 |
| **香港公司** | 以HK公司名义申报 | 中（CRS穿透控制人规则） | 中等（离岸豁免条件苛刻） | 中 | 中 |
| **BVI/开曼公司** | 以离岸公司名义，但CRS穿透控制人 | 高（BEPS要求经济实质） | 不确定（实质要求） | 中 | 中 |
| **离岸信托** | 受益人/控制人可能被穿透申报 | 视信托类型而定 | 中等 | 最高 | 最高 |

### CRS对法人结构的穿透规则

**关键风险**：金融机构在CRS尽职调查中，对于**被动非金融实体（Passive NFE）**，必须穿透识别"控制人"（Controlling Person）：

- BVI公司、开曼公司、普通香港控股公司 → 通常被认定为Passive NFE
- 金融机构会追问：谁是实际控制人？控制人的税务居民国是哪里？
- 如果控制人是中国税务居民 → 即使通过公司持有，信息仍会上报至中国

**相对安全的结构**：
- **新加坡SFO（获批的13O/13U基金）**：被认定为"金融机构"而非Passive NFE，不须穿透到个人层面（但基金自身须申报）
- **新加坡VCC**：同上
- **美国公司（LLC等）**：美国不参与CRS，即使被穿透，信息也不进入CRS交换体系

## 五、保险架构的CRS处理

### 亚洲储蓄险/ILAS在CRS下的申报

| 产品类型 | CRS申报要求 |
|---------|-----------|
| 有现金价值的储蓄险（如香港分红险、新加坡储蓄险） | ✅ 须申报（被视为特定保险合同） |
| 纯消费型寿险（Term Life，无现金价值） | ❌ 不须申报（豁免）|
| 年金产品（保证收益类） | ✅ 须申报 |
| 配合SFO持有的私募保险包装（PPLI） | 视结构而定，通常须咨询 |

### 私人配售保险（PPLI）
- 将投资资产"包装"在保险壳内
- 合法的税务递延/优化工具
- 在美国税法下有特殊处理（须满足多元化投资测试和投资者控制测试）
- 在拿美国绿卡前应完成PPLI的结构审查

## 六、家族信托的信息交换风险

### 全权信托（Discretionary Trust）的CRS处理

```
信托架构：
委托人（Settlor）→ 托管人（Trustee）→ 受益人（Beneficiaries）
```

**申报逻辑**：
- **信托资产账户**由受托人（Trustee）持有
- CRS要求：识别受托人、委托人（若有保留权力）、保护人和**受益人**的税务居民状态
- 全权信托中，若受益人为"可确定受益人"（Ascertainable Beneficiary），须披露

**信托隔离效果的局限性**：
- 若委托人是中国税务居民 → 仍可能被识别
- 若受益人是中国税务居民 → 须上报至中国
- 切断中国税务居民身份 + 信托架构 = 才能真正隔离

## 八、关键法规依据

| 法规 | 内容 |
|------|------|
| OECD CRS Standard（完整文本） | 金融账户申报标准；豁免账户类型；控制人穿透规则 |
| 《金融账户涉税信息自动交换管理办法》（2017年） | 中国CRS实施细则 |
| OECD CRS参与国列表（持续更新） | 哪些国家向中国交换数据 |
| FATCA IGA Model 1 & Model 2 | FATCA协议类型；SG、HK、CN均已签署 |
| BEPS Action 5 | 经济实质要求（离岸架构） |
| BEPS Action 6 | 协定滥用反避税规则 |
| OECD《CBI/RBI高风险项目报告》（2022） | CBI护照的CRS穿透风险 |
| IRC §871, §1441 | NRA在美账户的预提税规则 |
| FinCEN 114 / Form 8938 | 美国绿卡持有者的境外账户申报义务 |
