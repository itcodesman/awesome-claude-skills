# 司法辖区比较与选择框架

## 一、美国信托核心辖区

### 南达科他州（South Dakota） ⭐ 综合最优

| 特点 | 说明 |
|------|------|
| 税务 | 无州所得税、无遗产税、无特许经营税 |
| Dynasty Trust | 无存续期限制（可永续） |
| 资产保护 | 法律完善，等待期短 |
| 隐私 | 严格立法，不公开受益人信息 |

### 内华达州（Nevada）

- 无州所得税，遗产税友好
- 资产保护信托（APT）强度高，债权人等待期**仅2年**
- 商业友好，公司注册便利

### 德拉瓦州（Delaware）

- 公司法最为完善，LLC架构灵活
- 信托法律成熟，**Directed Trust制度**明确分权架构（分配顾问/投资顾问权力明确分离）

### 美国信托整体优势（vs 离岸）

1. **非CRS参与国**：无自动向中国交换账户信息义务
2. 受托人可直接以信托EIN在券商/银行开户，无需SPV嵌套
3. 法律体系成熟稳定，受托人资质可信度高
4. 资产保护与税务规划双重能力

## 二、离岸信托辖区

### 库克群岛（Cook Islands） ⭐ 资产保护最强

| 特点 | 说明 |
|------|------|
| 资产保护 | **全球最强**：不承认外国法院判决，债权人诉讼实际无效 |
| 欺诈转让 | 等待期仅适用于本地法院认定，海外债权人极难成功 |
| CRS状态 | 参与国 |
| 中国籍策略 | 非中国籍委托人设立可切断与中国的信息链 |
| 迁移性 | 架构可迁移性强 |

### 开曼群岛（Cayman Islands）

- 成熟的基金与信托法律体系
- CRS参与国，广泛用于**私募基金架构**
- 无法直接开户，须嵌套SPV

### 英属维尔京群岛（BVI）

- 灵活的公司与信托法律
- 广泛用于**持股结构中间层**
- CRS参与国，隐私保护已明显下降

### 巴哈马（Bahamas）

- 历史重要离岸信托辖区
- 2016年巴哈马文件泄露后，大量客户迁移至库克群岛
- 目前吸引力明显下降

## 三、欧洲结构选项

### 欧洲基金会（Foundation）⭐ CRS穿透防御最强

主要辖区：**列支敦士登、奥地利、荷兰、马耳他**

| 特点 | 说明 |
|------|------|
| 法律结构 | 大陆法体系，基金会为独立法人，**无所有人概念** |
| CRS穿透防御 | 最强：创始人（Founder）权益已切断，CRS报告链中断 |
| 受益人隐私 | 极强 |
| 治理结构 | 创始人 → 基金会委员会 → 受益人 |
| 年度维护费 | 资产规模1%以上，成本较高 |

### 信托 vs 基金会 对比

| 维度 | 信托（Trust） | 基金会（Foundation） |
|------|-------------|-------------------|
| 法律体系 | 普通法 | 大陆法 |
| 所有权归属 | 受托人（法定）/ 受益人（衡平） | 基金会（独立法人，无所有人） |
| CRS穿透风险 | 中等（委托人为实控人） | 低（创始人权益已切断） |
| 设立成本 | 中等 | 较高 |
| 适用国家 | 英美法系国家更友好 | 欧洲大陆及大陆法系国家 |
| 年度维护 | 较低 | 较高（约资产1%） |

## 四、CRS参与状态对辖区选择的影响

> 核心逻辑：**美国是CRS非参与国**，这是其相较于香港/新加坡的最大优势之一。

| 辖区 | CRS状态 | 对中国籍委托人的影响 |
|------|---------|-------------------|
| **美国** | **非参与国** | ✅ 不向中国自动交换信息，安全性高 |
| 香港 | 参与国 | ⚠️ 信托被穿透，实控人信息报送至中国 |
| 新加坡 | 参与国 | ⚠️ 同上，误信可规避CRS的架构已被大量拆除 |
| 开曼/BVI | 参与国 | ⚠️ 同上 |
| 库克群岛 | 参与国 | 非中国籍委托人设立可切断中国信息链 |
| 欧洲基金会辖区 | 多数参与 | 基金会结构本身可中断CRS报告链 |

---

## 五、辖区选择决策树

```
目标：CRS隐私 + 资产保护 + 税务优化？
│
├─ 是否接受美国（非CRS）结构？
│   ├─ 是 → 南达科他州不可撤销信托（综合最优）
│   └─ 否（需要离岸）→ 资产保护优先级？
│       ├─ 最高 → 库克群岛APT（须以非中国籍身份设立）
│       └─ 隐私穿透防御优先 → 欧洲基金会（列支敦士登/马耳他）
│
└─ 现有架构在香港/新加坡？
    → 强烈建议迁移，CRS已无法规避
```
