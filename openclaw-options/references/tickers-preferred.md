# 全量池（preferred / full）参数手册

> **适用范围**：`m.scope == "preferred"` 或 `m.scope == "full"`。
> **与 20tickers 的关系**：同一标的（如 TSLA/AAPL/HOOD）若在 20tickers 表与本表参数不一致，以 `m.scope` 对应的表为准；不做跨表合并。详细路由见 `references/tickers.md`。
> **扩展池**：`off` 键不存在的标的为扩展池参数推断，推送时必须注明"扩展池·参数推断，需人工确认"。

## 一、全量池特殊规则（追加硬约束）

> 速查总表已含基础参数；此处仅列超出通用规则的**硬约束**。

**SPY / QQQ**：合计 ≤ 净值 25%，不同时满仓；QQQ 在 NVDA/AAPL 财报周若个股已开仓，仓位 ×0.7。

**EWJ**：净值 ≥ $500K；USD/JPY 当周变动 <3% 自动暂停；BOJ 前后 2 天停；**C 状态直接暂停，不做价差**。

**AAPL（全量模式）**：被行权 3 交易日内强制推 CC，行权价 ≥ 持仓成本；接货后正股市值超 $60,000 → 推遗产税警告。

**NVO**：FDA 关键评审日视同财报；重大 GLP-1 竞争负面事件推暂停。

**ASML**：**C 状态直接暂停**；买卖价差/中价 <4%；净值 ≥ $1M；出口管制新闻当日暂停。

**TSM**：**C 状态直接暂停**；关键词"台海"/"TSMC ban"/"chip export"触发当日减仓 50% + 暂停新开仓。

**GLD**：满足任一才推：① VIX >22；② 地缘危机标记=TRUE；③ GLD 单日涨 >1.5% 且 VIX 同日涨 >10%。**VIX<18 且无地缘触发禁推**。

**XLU**：FOMC 决议日前 5 交易日停开（比通用 3 天严）；C 状态允许，仓位 ×0.5，Delta 取矩阵下限。

**META**：持有 META 正股时禁止同时卖 Put；广告收入/DAU 季度预报视同财报前 14 天。

**NVDA**：BIS 出口管制变动当日暂停；**NVDA + TSM + ASML ≤ 净值 15%**。

**TSLA（全量模式）**：仅状态 A；TSLA > 200MA；Musk 声明前 14 天停。合约张数上限见 `SKILL.md §PM-1`。

**HOOD（全量模式）**：BTC 跌 >8% 停；OI ≥ 5,000 张。合约张数上限见 `SKILL.md §PM-1`。

**IBIT**：BTC 现货 > 50 日均线；SEC 监管动作推仓位评估警告。

**COIN（全量模式）**：仅状态 A；COIN > 200MA；BTC > 50MA；与 MSTR 互斥；单标 ≤ 净值 2%。

**MSTR**：仅状态 A；BTC > 50MA **且 > $60,000**；与 COIN 互斥；单标 ≤ 净值 1.5%；跌破任一条件立即推"评估平仓"。合约张数上限见 `SKILL.md §PM-1`。

**CRCL**：上市满 180 天后才进扫描池；OI ≥ 5,000 张；稳定币监管立法/执法 → 最高级风险警告；单标 ≤ 净值 1%。

**MARA**：BTC 1:1 联动；合约张数上限见 `SKILL.md §PM-1`。

## 加密组合总控（指针）

全量池涉及 `IBIT` / `COIN` / `MSTR` / `HOOD` / `CRCL` / `MARA` 等。

**唯一权威定义**：`SKILL.md §PM-1 · 加密子条款`（≤ NLV × 4%，优先级高于 PM-1 单标 10% 上限）。本文件各标的的"单标 ≤ 净值 X%"是在 4% 总控内进一步收紧，两者同时约束、取更严值。
