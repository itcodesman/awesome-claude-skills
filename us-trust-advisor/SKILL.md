## 核心分析框架

在回答任何信托架构问题前，先确认用户的**核心目标优先级**：

```
1. 资产保护（债权人隔离、破产隔离）
2. 财富传承（跨代延续、受益人管控）
3. 遗产税规划（减少遗产税敞口）
4. CRS/信息隐私（切断与中国的信息链）
5. 控制权保留（间接控制资产运作）
```

不同目标对应不同辖区和架构选择——这是所有建议的出发点。

## 核心概念速查

| 中文 | 英文 | 核心含义 |
|------|------|---------|
| 委托人 | Settlor / Grantor | 信托创设者，财产转移方 |
| 受托人 | Trustee | 法定财产持有者，信托执行者 |
| 监察人 | Protector | 制衡核心，可更换受托人 |
| 分配顾问 | Distribution Advisor | 分配执行的否决权持有人 |
| 受益人 | Beneficiary | 信托收益享有方 |
| 信托协议 | Trust Deed | 信托基本法律文件，修改成本高 |
| 意愿书 | Letter of Wishes | 动态管理工具，条件性分配指引，non-binding |
| 不可撤销信托 | Irrevocable Trust | 资产脱离委托人个人财产 |
| 可撤销信托 | Revocable Trust | 资产仍属委托人个人财产 |
| 资产保护信托 | APT | 专门抵御债权人追索 |
| 私人信托公司 | PTC | 家族自设受托人，隐私性最强 |
| 子信托 | Sub-Trust | 单一资产对应单一风险隔离单元 |
| 人寿保险信托 | ILIT | 持有保单，保险理赔不计入遗产 |
| 永续信托 | Dynasty Trust | 无存续期限制，跨代延续 |
| CRS | Common Reporting Standard | 全球金融账户信息自动交换标准（OECD） |
| FATCA | Foreign Account Tax Compliance Act | 美国对外籍账户的税务合规要求 |
| EIN | Employer Identification Number | 美国信托独立税号，可直接开券商/银行账户 |
| KYC/AML | Know Your Customer / Anti-Money Laundering | 身份核查与反洗钱 |
| RIA | Registered Investment Advisor | 美国注册投资顾问牌照 |
| SFO/MFO | Single/Multi Family Office | 单一/联合家族办公室 |
| OFAC | Office of Foreign Assets Control | 美国财政部制裁名单管理机构 |
| 欺诈性转让 | Fraudulent Conveyance | 在债务压力下转移资产，可被法院撤销 |
| 等待期 | Look-back Period | 欺诈性转让追溯期，通常2–4年 |
